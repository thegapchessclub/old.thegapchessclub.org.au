Goals achieved:

  • Proposed web site games page operational.
  • 81 games uploaded.
  • Almost 400 visits to the Flood Cup and One Hour games pages.

 

 

Goals not achieved :

  • Web site game generator unavailable for public use. Still in a non user friendly state.
  • Proposed Chocolate Drive Fundraiser. There is a requirement from Cadbury that there be a minimum purchase of 12 boxes at a total cost of $288 (50c per chocolate). This will require people from within the club to volunteer to sell a box at their workplace. Recommended sale price of each chocolate is $1, so the break-even point will be achieved once half of the chocolates are sold. Need to discuss the risk factor involved with such a large upfront payment.

 

 

Goals for the next year:

  • Web site games pages to receive some enhancements.
  • A few more viewing options, which includes the animation of pieces.
  • The ability to move pieces, allowing the viewer to take over a game at a particular point to do some post-mortem analysis.
  • The ability for anyone to enter their own games via the web site. They can generate and send these for the site admin to upload. Method for entering will be either by movement of pieces, or by typing in the notation text, or perhaps both.
  • The possible adoption of the popular .pgn format to allow for generated games to be used in other chess programs.
  • To update the pre-2004 games on the web site to the graphical format.
